# FAQ

Q: Where to start?
A: Use one of:
- `custom/docs/tool-project/README.md` (maintainers)
- `custom/docs/target-project/README.md` (target project users)

Q: Where is project-specific failure knowledge stored?
A: In the target project at `.multipowers/FAQ.md`.

Q: Do I need to maintain `.multipowers/FAQ.md` manually?
A: No. It is automatically generated, deduplicated, and rewritten by the multipowers FAQ synthesizer.
